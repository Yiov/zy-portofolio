## 步骤

> 注意：如果你拉的是我的仓库，就不需要安装 `npm-check-updates`

```
#安装ncu
npm i -g npm-check-updates

#更新依赖包版本
ncu -u



#安装依赖
pnpm install

#开发
pnpm run serve

#构建
pnpm run build
```
